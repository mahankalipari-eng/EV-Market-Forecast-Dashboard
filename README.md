# EV Market Forecast Dashboard

## Objective
To analyze historical Electric Vehicle (EV) sales data and predict future market growth using Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Features
- Historical EV Sales Analysis
- EV Market Forecasting (2026-2030)
- Growth Rate Calculation
- Data Visualization using Graphs

## Results
Predicted EV sales from 2026 to 2030 and estimated market growth rate of 40.28%.
